# 文件夹文件管理插件

一个用于AstrBot的文件夹文件管理插件，可以监控指定文件夹并支持文件列表查看和文件发送功能。

## 功能特性

- 📁 **文件列表查看**：列出监控文件夹中的所有文件
- 📤 **文件发送**：通过编号发送指定文件
- 🔄 **缓存管理**：智能缓存文件列表，减少文件系统读取
- ⚙️ **配置灵活**：支持文件类型过滤和大小限制
- 🚀 **平台优化**：针对不同消息平台优化文件发送

## 安装方法

1. 将插件文件夹放置到 `AstrBot/data/plugins/` 目录下
2. 重启AstrBot或重载插件
3. 在WebUI中配置插件参数

## 使用方法

### 基本命令

- `/listfiles` - 列出文件夹中的所有文件（带编号）
- `/sendfile <编号>` - 发送指定编号的文件
- `/refreshfiles` - 强制刷新文件列表（忽略缓存）
- `/fileinfo` - 显示文件夹统计信息


## 配置说明

在插件配置页面可以设置以下参数：

- **监控文件夹路径**：要监视的文件夹路径
- **缓存时间**：文件列表缓存的有效时间
- **允许的文件扩展名**：只显示指定类型的文件
- **最大文件大小**：限制可发送的文件大小
